## Функция для формирования .xlsx из .md

# parser.py:
Функция `parse_md` парсит .md файл и возвращает список задач. Распознает:
- Заголовки вида \title{ЧАСТЬ N} / \section*{ЧАСТЬ N} — многострочно или в одну строку
- Номера задач вида А1, В1, С1 - латиницей и кириллицей
- Ссылки на изображения в формате ![](url)
- Остальной текст записывается в текущую задачу

# exporter.py:
Функция `export_xlsx` получает список словарей и записывает список задач в .xlsx файл в соответствующие колонки.

# main.py:
Функция `convert` конвертирует .md в .xlsx с тем же именем файла.

##  Запуск:

Создать и активировать виртуальное окружение из папки MARKDOWN_TO_EXCEL:

`` 
python -m venv .venv 

source .venv/Scripts/activate 
``` 
Установить зависимости из файла requirements.txt: 
``` 
pip install -r requirements.txt 
``` 
Запуск командой:
``` 
python main.py 'file_name.md'
```
